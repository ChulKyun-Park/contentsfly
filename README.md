# ContentsFly

영상 번역 전문 플랫폼 — 전문가 번역부터 AI 번역·더빙까지, 크리에이터와 기업의 글로벌 진출을 지원합니다.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Tech Stack

- Next.js 16 + React 19 + TypeScript
- Tailwind CSS 4
- Notion API (CMS)
- Vercel (Deploy)
